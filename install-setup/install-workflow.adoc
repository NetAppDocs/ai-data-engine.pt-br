= Fluxo de trabalho de instalação e configuração para nós de computação de dados para AI Data Engine
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para instalar e configurar seus nós de computação de dados (DCN), você revisa os requisitos de hardware, prepara seu local, instala e conecta os componentes de hardware, liga o sistema e configura seu cluster ONTAP.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Um"]link:install-setup-requirements.html["Revise os requisitos de instalação do hardware"]
[role="quick-margin-para"]
Certifique-se de que você já possui um sistema de storage AFX 1K instalado e, em seguida, revise os requisitos de hardware para instalar os nós de computação de dados para AIDE. Para obter informações sobre a instalação do sistema de storage AFX 1K, consulte link:https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["Documentação de instalação do sistema de storage AFX 1K"^].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dois"]link:prepare-hardware.html["Prepare-se para instalar seus nós de computação de dados"]
[role="quick-margin-para"]
Para se preparar para instalar seus nós de computação de dados, você precisa organizar o local, verificar os requisitos ambientais e elétricos e garantir que haja espaço suficiente no gabinete. Em seguida, desembale o equipamento, compare o conteúdo com o packing slip e registre o hardware para acessar os benefícios de suporte. São necessários no mínimo três nós de computação de dados para implantar o AI Data Engine.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Três"]link:deploy-hardware.html["Instale o hardware para seus nós de computação de dados"]
[role="quick-margin-para"]
Instale os kits de trilhos para seus nós de computação de dados. Fixe seus nós de computação de dados dentro do gabinete. Por fim, conecte os dispositivos de gerenciamento de cabos na parte traseira do sistema para organização dos cabos.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatro"]link:cable-hardware.html["Conecte seus nós de computação de dados"]
[role="quick-margin-para"]
Para conectar o hardware, primeiro conecte os nós de computação de dados à sua rede de dados e depois conecte os nós de computação de dados aos switches do cluster.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinco"]link:power-on-hardware.html["Ligue seus nós de computação de dados"]
[role="quick-margin-para"]
Após instalar o hardware do rack e conectar os nós de computação de dados, você deve ligar os DCNs e os nós controladores do sistema de storage AFX, caso ainda não estejam ligados.
