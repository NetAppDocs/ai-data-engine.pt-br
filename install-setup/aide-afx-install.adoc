= Instale o sistema de storage AFX para AI Data Engine
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Instale o sistema de storage AFX como o primeiro passo na implantação do AI Data Engine (AIDE). O sistema de storage AFX fornece a base da infraestrutura de storage e é necessário antes de instalar os nós de computação de dados.

Siga o link:https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["Documentação de instalação do AFX 1K"^] para instalar o sistema de storage AFX.

.O que vem a seguir
Após concluir a instalação do sistema de storage AFX, link:install-workflow.html["instale os Data Compute Node"].
