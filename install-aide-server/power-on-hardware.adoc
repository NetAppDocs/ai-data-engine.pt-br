= Ligue seus servidores de terceiros para AI Data Engine
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Após instalar o rack e conectar os cabos do hardware, você deve ligar os servidores de terceiro e os nós controladores do sistema de storage AFX, caso ainda não estejam ligados.

Para servidores de terceiros, consulte a documentação de instalação do seu modelo específico para obter instruções de inicialização.
